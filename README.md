**# prompt-chaining
Intelligent Customer Support Prompt Chain**

This project implements a five-step prompt chain that simulates how an AI customer support system understands and responds to customer queries for a bank.

**Prompt Chain Stages**

Interpret Intent – Understand what the customer is asking or reporting.

Map to Possible Categories – Suggest all categories that might fit.

Choose Most Appropriate Category – Select the best matching category.

Extract Additional Details – Identify relevant information such as amounts, card types, or dates.

Generate Short Response – Produce a professional support message.

**Available Categories**

Account Opening

Billing Issue

Account Access

Transaction Inquiry

Card Services

Account Statement

Loan Inquiry

General Information

⚙️ How to Run
python3 prompt-chain.py
